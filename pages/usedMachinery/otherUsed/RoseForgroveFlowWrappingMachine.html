Rose Forgrove Flow Wrapping Machine

/ Description /


See Video

This used Rose Forgrove flow wrapper comes with many extras....

It has a driven film unwind, It has a roller system through the jaws

This ensures that products can never fall down between the sealing knives at this point

It is fitted with a 400 mm sealing jaw (adjustable height sealing jaws)

This machine is also fitted with a fully adjustable forming box (height and width)

The machine has only ever been used for packing computer software so it is very clean

This machine also has print registration which means it gives you the option of running printed film going forward


It is ideal for businesses who wish to wrap some different size products (would be ideal for a small bakery in our opinion)
