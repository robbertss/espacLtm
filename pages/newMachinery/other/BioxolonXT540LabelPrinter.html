Bioxolon XT5-40 Label Printer

/ Description /


The XT5-40 4-inch (114mm) is a Thermal Transfer high performance industrial desktop label printer ideal for high volume label printing in a variety of applications including Logistics, Manufacturing and Warehousing. Available as XT5-40 (203dpi), XT5-43 (300dpi) & XT5-46 (600dpi) print resolution media options, the printer’s standard model (XT5-40) supports fast data processing and print speeds up to 14ips (356mm/sec) to produce high quality text, graphics and 1D/2D barcodes with a 512 MB SDRAM, 512 MB Flash memory

Compatible with market-leading programming languages including SLCS, BPL-Z™, BPL-E™ and BPL-D™. The Industrial label printer enjoys optional built-in Bluetooth, Wi-Fi, Parallel, USB, Serial, Ethernet (the industry’s fastest 1Gbit/sec) and two USB Host interfaces options. Using the SoftAP feature it enables simple, fast and convenient Wi-Fi set up directly to a mobile device straight out of the box. The XT5-40 also features a Printer Twin Function™ to easily copy the printer’s settings and data (Templates, images and fonts) to other printer via USB storage

Alongside XPM™, BIXOLON’s web-based printer remote control profile manager and compatibility with label design software including BIXOLON’s Label Artist™, Label Artist™ Mobile (for iOS and Android) and BarTender® UltraLite. Plus the option of a customized stand-alone functionality, taking input from barcode scanners and other peripherals without any PC or network connection

Built to last in the harshest of environments, the XT5-40 supports a space saving design with a double folding door. An aluminum body with special anti-adhesive metal casing and a 3.5-inch full colour TFT LCD display with intuitive UI. Supporting a media roll diameter up to 203mm, ribbon length up to 450mm and adjustable print head pressure

The printer has an intuitive design allowing easy maintenance of the roller and printhead and a single screw interchangeable Auto-Cutter, Peeler, Rewinder, or Rewinder + Peeler to give users the chance upgrade as their printer as their requirements evolve
